# Telebirr Receipt Clone

A pixel-perfect replica of the Telebirr transaction success receipt.

## Features

- Authentic Telebirr UI design
- Samsung M1 & iPhone XR frame support
- Interactive "Save as Image" functionality
- Pixel-perfect QR Code row calibration

## Tech Stack

- React
- Vite
- Tailwind CSS
- html-to-image
- Lucide React
